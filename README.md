# AI-Powered Recipe Recommendation System

The AI-Powered Recipe Recommendation System is an intelligent web application that combines Machine Learning, Natural Language Processing (NLP), Semantic Search, and Generative AI to provide personalized recipe recommendations and customized meal suggestions. The system analyzes user preferences, dietary restrictions, allergies, nutritional goals, favorite cuisines, and ingredient choices to generate highly relevant recipes tailored to individual needs.

The recommendation engine integrates Content-Based Filtering, Collaborative Filtering, Sentence-BERT (SBERT) embeddings, and FAISS vector search to identify recipes that closely match user interests. In addition, Generative AI is used to create personalized recipe suggestions, ingredient substitutions, meal ideas, and cooking recommendations based on user requirements.

## Features

### AI & Machine Learning Features

* Personalized AI-Powered Recipe Recommendations
* Generative AI-Based Recipe Suggestions
* Hybrid Recommendation System (Content-Based + Collaborative Filtering)
* SBERT Semantic Similarity Search
* FAISS Vector-Based Recipe Retrieval
* Intelligent Recipe Ranking and Matching
* Context-Aware Food Recommendations
* Preference Learning and Personalization

### User Features

* User Registration and Authentication
* Personalized User Profiles
* Preference-Based Recommendations
* Favorite Recipe Management
* Recommendation History Tracking

### Dietary & Health Features

* Vegetarian, Vegan, and Non-Vegetarian Support
* Allergen-Aware Filtering
* Health-Conscious Meal Recommendations
* Nutrition-Based Recipe Selection
* Dietary Restriction Management

### Search & Discovery

* Ingredient-Based Search
* Cuisine-Based Search
* Meal-Type Filtering
* Semantic Recipe Search
* Intelligent Recipe Discovery

### Recipe Information

* Detailed Cooking Instructions
* Ingredient Lists
* Nutritional Information
* Preparation and Cooking Time
* Recipe Categories and Tags

### Technology Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python, Flask
* Database: Food.com Recipe Dataset
* Machine Learning: Scikit-learn
* NLP: Sentence-BERT (SBERT)
* Vector Search: FAISS
* AI Integration: Generative AI for recipe generation and personalization
* Data Processing: Pandas, NumPy

## Objective

The goal of this project is to enhance recipe discovery and meal planning through Artificial Intelligence by delivering personalized, health-aware, and context-sensitive recipe recommendations. The system helps users find suitable recipes quickly while adapting to their tastes, dietary needs, and nutritional requirements.
